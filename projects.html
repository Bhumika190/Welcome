PROJECTS

URL SHORTNER

I developed a URL Shortener application using Java to convert long URLs into short, unique, and easy-to-share links. The project was built using a layered architecture approach, where I implemented RESTful APIs to generate, store, and redirect shortened URLs efficiently. I used Java (Spring Boot) for backend development and integrated a database (such as MySQL/MongoDB) to persist original and shortened URLs along with metadata like creation time and click count. The application generates unique short codes using encoding techniques and ensures no duplication through proper validation mechanisms. I also implemented redirection logic so that when users access the short URL, they are automatically redirected to the original link. Additionally, I focused on handling edge cases, improving performance, and maintaining clean code structure following best practices. This project strengthened my understanding of backend development, API design, database integration, and real-world problem-solving using Java.

TWITTER CLONE

I developed a Twitter Clone using Java and modern web technologies to replicate the core functionalities of Twitter (now known as X). The application allows users to register, log in, create posts (tweets), like and retweet content, follow or unfollow other users, and view a personalized news feed. I implemented a clean and scalable backend architecture using Java (Spring Boot), following best practices such as layered architecture and RESTful API design. The project includes modules for user management, tweet handling, follower relationships, notifications, and search functionality. I integrated a relational database to manage user data, posts, and interactions efficiently, ensuring proper entity relationships and optimized queries. Authentication and authorization were handled securely using JWT-based authentication. Additionally, I focused on writing maintainable code by applying design principles like SOLID and implementing dependency injection. This project enhanced my understanding of backend system design, real-time social media features, database modeling, and building scalable web applications using Java.

ONLINE SHOPPING

I developed an **Online Shopping Application** using Java to simulate the core functionalities of modern e-commerce platforms like Amazon and Flipkart. The application allows users to register, log in, browse products by category, add items to the cart, place orders, and view order history. I implemented a layered architecture using Java (Spring Boot) and designed RESTful APIs to handle product management, cart operations, user authentication, and order processing. A relational database was integrated to manage users, products, orders, and payment details with proper entity relationships and validations. I also implemented secure authentication and authorization mechanisms to protect user data and ensure role-based access (Admin and Customer). Additionally, the admin module enables product addition, updates, deletion, and inventory management. This project strengthened my understanding of backend development, database design, transaction handling, API development, and building scalable real-world e-commerce systems using Java.
